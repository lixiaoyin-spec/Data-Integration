## 二、数据表结构说明

### 1. 学生表（student）

| 字段名          | 类型          | 说明   |
| ------------ | ----------- | ---- |
| student_id   | varchar(12) | 学号   |
| student_name | varchar(10) | 姓名   |
| gender       | varchar(2)  | 性别   |
| department   | varchar(16) | 院系名称 |
| account      | varchar(10) | 登录账号 |
| password     | varchar(6)  | 登录密码 |
| group_no     | varchar(10) | 组号   |
| dept_no      | varchar(10) | 院系编号 |

------

### 2. 课程表（course）

| 字段名            | 类型          | 说明     |
| -------------- | ----------- | ------ |
| course_id      | varchar(8)  | 课程编号   |
| course_name    | varchar(16) | 课程名称   |
| credit         | varchar(2)  | 学分     |
| teacher_name   | varchar(20) | 任课教师   |
| location       | varchar(20) | 上课地点   |
| share_flag     | char(1)     | 是否共享课程 |
| class_hours    | varchar(10) | 理论学时   |
| practice_hours | varchar(10) | 实践学时   |
| group_no       | varchar(10) | 组号     |
| dept_no        | varchar(10) | 院系编号   |

------
